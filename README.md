## 🚀 Getting Started

### Clone

```sh
git git@github.com:vieiramanda11/mediCo-API.git
cd medico-api
```

### Install

```sh
bundle install --without production
```

### Usage

```sh
rails db:migrate
rails db:seed
rails server
```

### Run tests

```sh
rspec
```
